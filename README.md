## 需求分析
#### 功能性需求
1.  输入一张通用格式的图片(.jpg .png .gif .tiff)，在桌面输出图片识别结果
2.  对于识别结果，如果不是母语中文，自动翻译成中文（如果做得完的话
3.  支持登录注册，用云端存储用户信息。
4.  支持管理员对用户信息和权限的修改
5.  系统支持悬浮操作，可仅用鼠标完成识别功能

#### 非功能性需求
1.  单次识别结果响应时间在3s内
2.  系统支持识别的图片最大体积至少超过300KB
3.  使用数据库保存用户信息以及设定
4.  若用户担心个人隐私，则使用本地OCR识别（如果可以顺带加上的话
5.  兼容Windows10 和Windows 11，可移植性较好
